# 沢城みゆき生日快乐！

6月2日是沢城みゆき的生日。

沢城みゆき于1999年以Petit Charat的角色在《Dejikara》中出道。她曾多次获得第3届声优奖的女配角奖、第4届的女主角奖、第5届的海外粉丝奖和第9届的女配角奖。

2023年，她将在电视动画、电影等各种作品中出演《宇宙猫忍者小町》、《金之国 水之国》、《鬼太郎诞生 妖怪哥哥的谜团》等。

为了庆祝沢城みゆき的生日，我们进行了题为“你最喜欢的角色是谁？”的读者调查。在5月17日至5月24日的调查期间，我们收到了541份回答。男女比例约为40%男性和60%女性，女性略多。年龄层以19岁以下约占40%、20岁代约占30%为主。

# 美丽和强大并存的角色位居前列！

## 第一名

第一名是《鬼灭之刃》的堕姬，支持率约为13％，连续两年获得第一名。

今年在电影院上映的《鬼灭之刃：无限列车篇》中，堕姬作为重要角色出现，她残忍而妖娆，同时也拥有兄妹之爱，这个复杂的角色正是由于沢城美雪的表现力而得以成立。她完美地演绎了蕨姬花魁的残忍、上弦之陆的强大和可怕、妓夫太郎的妹妹气质以及最后梅的坚强。许多粉丝被她的表演所感动，其中有人说：“堕姬的情感变化在沢城美雪的表演中得到了传达。特别是在“哥哥！”这一场景中，我也跟着哭了。”

## 第二名

第二名是《刀剑神域》的Sinon，支持率约为12％，与去年相同。

她用纤细的身体巧妙地操纵着50口径的大口径步枪，让人惊叹。尽管在游戏中表现出强大的实力，但在现实中她却有着心理创伤，她克服这些创伤的坚强表现也很酷。有人说：“在《刀剑神域》中，Sinon是一个冷静的角色。沢城美雪的冷静声音是最好的搭配。当她的角色被Kirito抓住尾巴时，她发出的尖叫声非常可爱。”
