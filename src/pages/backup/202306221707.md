# 変な家朗读剧将于2023年7月在东京上演

由千叶翔也、神尾晋一郎、土田玲央、堀江瞬、榊原优希、石见舞菜香、八神莲、太田梦莉等声优和演员共同出演的朗读剧《変な家》将于2023年7月29日至30日在东京的“剧场1010”上演。这是神秘的面具作家和YouTuber“雨穴”的处女作品的朗读剧化。演员阵容也发表了评论。

YouTuber雨穴是一位拥有超过90万订阅者的恐怖小说家，以10到20岁的年轻人为主要受众。他的黑色全身紧身衣和白色面具非常引人注目，加上隐藏了本名、真面目和真实声音，更增加了他的神秘感和话题性。

《変な家》是基于2020年在网络媒体“Omocoro”上发表的文章，由雨穴在自己的YouTube频道上发布的房地产悬疑故事。从某个房屋的“平面图”被提交给超自然专栏作家“作者”，揭开了一个可怕的、不愿相信的事实和隐藏的秘密之门。由于其与现实紧密相连的生动性，在网络上广泛传播，视频版在2023年6月时已经被观看超过1300万次。

![変な家](https://animeanime.jp/imgs/zoom/593489.jpg)
2021年，由飞鸟新社出版的小说处女作品引起了轰动，被称为“一旦开始阅读就停不下来”，目前已突破60万部。2023年，该作品将在一迅社创办的WEB漫画新标签“comic HOWL”上进行漫画化，而且还决定于2024年拍成电影等，从一篇文章中展开了各种媒体混合展开。

这次的朗读剧将在东京的“剧场1010”举行，为期两天，共四场演出，演员阵容汇集了人气声优和演员。由土田玲央、堀江瞬、千叶翔也、八神莲、神尾晋一郎和榊原优希扮演的“作者”是一名专门从事神秘学的自由撰稿人，“作者”的朋友“栗原”则由八神蓮、神尾晋一郎和榊原優希扮演。此外，石见舞菜香和太田夢莉也加入了演员阵容，观众可以享受到由不同演员组合带来的变化。此外，导演村井雄曾执导《北斗之拳-世纪末Zako传说-》和《SK∞ Skate The Stage The Last Part～我们的无限大～》等作品。

由千叶翔也、神尾晋一郎、土田玲央、堀江瞬、榊原優希、石見舞菜香、八神蓮和太田夢莉出演的朗读剧《奇怪的房子》将于2023年7月29日至30日在东京的“剧场1010”上演。这是一个从一个平面图开始的令人毛骨悚然的故事，其结局一定会让人震惊。请关注这种与视频、小说和漫画不同的新型表现形式的朗读剧。

# 栗原角色的出演让我很高兴

我很高兴能够在这个朗读剧中扮演栗原的角色。这是一部推荐给喜欢推理小说的人，甚至是不擅长推理的人的作品。如果你觉得“阅读有点困难……”，那么为什么不尝试在朗读剧中体验新的推理体验呢？

# 筆者角色：千叶翔也

在有限的人数中进行朗读，这种紧张感，能够与两位主演一起演出，我感到非常荣幸和高兴。个人而言，我喜欢世界历史，对世界上的谜团和城市传说等话题深入探究，所以我认为对这个话题感兴趣的人会很多。

虽然我不能完全想象出房间的布局等细节，但我会尽力通过演员的声音来传达。即使你对“奥卡尔特”和“房地产”这些词汇不熟悉，也请放心前来参加，我们在会场等待着你。

# 栗原角色：榊原优希

实际上，我一直喜欢在互联网上寻找奇怪的房间布局或特殊的房子，通过这种方式，我在得到这个朗读剧的邀请之前就已经找到了这个故事。这个故事从一个看起来真实的房间布局开始展开，具有真实感和令人愉悦的推理。
# 【宫江柚希 饰】太田梦莉
